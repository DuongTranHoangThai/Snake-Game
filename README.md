# Snake-Game

Linux terminal Snake game 

**Compile:**
gcc Snake.c -o Snake

**Run:**
./Snake

**Modify width and height game border**
Change value of
#define WIDTH 40
#define HEIGHT 20

**Used technique:**
- ANSI escape command (to control the terminal cursor, hide the cursor)
- Snake game algorithm (available on Google)
